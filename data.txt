## Projects

### AgentCommerce — Agentic Society Hackathon, April 2026 (3rd place, £2,000 prize)

AgentCommerce is a decentralised e-commerce platform that connects AI agents to real Shopify stores via the Hedera blockchain. A vendor installs a single Shopify plugin, which publishes their entire product catalogue to a blockchain registry. An AI shopping agent then interprets a user's intent (for example, "buy batteries"), searches the registry for the best price, and executes a real HBAR cryptocurrency payment — all autonomously. The order is submitted back to the vendor's Shopify for fulfilment. Daniyal built the AI shopping agent and the backend integration layer using Node.js, Express, and the Hedera SDK. The judges praised the end-to-end integration: not just an agent that talks, but one that spends real money and triggers real fulfilment. GitHub: https://github.com/Daniyalsid26/agentic-ai-hack

### Formal Verification of Python Code using Why3 — UOG CS2, 2025

Daniyal built an automated LangGraph framework to formally verify Python code against Why3 mathematical standards. The system uses a graph architecture where an LLM calls tools, parses outputs, and loops with Human-in-the-Loop checkpoints until the code satisfies formal correctness constraints. Key techniques: ReAct tool calling, agentic memory, and iterative prompt engineering. GitHub: https://github.com/mtaimoorkhan/TrustedCode — Video: https://youtu.be/Nr5I2g8ec-s

### ClimateImpact Hackathon — 1st place for Impact, June 2025

Daniyal's team placed 1st for Impact out of 15 teams. They built an AI assistant designed to evaluate and de-risk agricultural loans for farmers, helping lenders make more informed decisions in climate-vulnerable regions.
